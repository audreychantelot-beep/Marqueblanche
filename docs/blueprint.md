# **App Name**: Marque blanche

## Core Features:

- Sidebar Navigation: Animated sidebar navigation with collapsible links for desktop and a mobile-friendly menu.
- Link Components: Reusable link components with animated text display based on sidebar state.
- Theme Toggle: Enable light/dark theme of the Dashboard

## Style Guidelines:

- Beige: #f4f4f4
- Blanc: #FFFFFF
- Noir: #000000
- Font pairing: 'Poppins' (sans-serif) for headlines and short amounts of body text, 'Inter' (sans-serif) for longer paragraphs.
- Use minimalist line icons from Lucide to maintain a clean and modern look.
- The sidebar should take up 300px of screen width. When the sidebar collapses, it should take up 60px.
- Use smooth transitions and subtle animations to enhance the user experience when toggling the sidebar and navigating between links.
